This action is used to guide you through setting up the Trådfri plugin.

Click on the detailed instructions in the below section.
Setup Instructions

1. Find the secret key located on the bottom side of your Trådfri gateway.

2. Press this action button on your Stream Deck to open the settings folder.

3. Open settings.json in a text editor.

5. Add the secret key Ex.: "GatewaySecret":"yourkeyhere"

6. Add an app name Ex.: "AppName":"SD" (An app name can only be used once!)

7. Press this action button again to generate the app key.

8. You should now see the IP address and firmware version of your gateway.

Visit the project page on GitHub for more information and troubleshooting.
